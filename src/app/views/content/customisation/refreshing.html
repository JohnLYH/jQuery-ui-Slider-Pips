Refresh Method

If the slider's options have change programmatically, then we probably need to update the pips/floats to reflect this.

Pips
Float
                // initialise the slider, and pips first
                $(".slider")
                    .slider({ max: 10 })
                    .slider("pips");

                // then change and option, and refresh the pips
                $(".slider")
                    .slider("options", "max", 20 )
                    .slider("pips", "refresh");
            
                // initialise the slider, and float first
                $(".slider")
                    .slider({ max: 10 })
                    .slider("float");

                // then change and option, and refresh the float
                $(".slider")
                    .slider("options", "max", 20 )
                    .slider("float", "refresh");
